Resumo do Projeto

nós somos os ratos da uninove e iremos apresentar nosso projeto que estivemos trabalhando. Em toda a nossa vida temos tarefas a serem compridas e ter organização sobre elas é essencial, pensando nisso que criamos o GREEN ORGANIZER, um ótimo gerenciador de tarefas que lhe permite criar tarefas simples de uma lista de checagem. Para essas listas usamos Javascript, HTML e Css para personalizar de forma dinâmica e ágil as tarefas, podendo organizá-las em, por exemplo, tarefas acadêmicas, pessoais, físicas etc. Para a criação do Back-end, após uma análise detalhada, considerações sobre MySQL, decidimos utilizar um banco de dados desenvolvido em C#. Com essa decisão, iniciamos a construção e a estruturação do banco de dados (DB)
